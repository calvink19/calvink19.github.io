- Cracked Archive
- General Archive
- BigBoss Archive
- EatMyPhone Archive

These aren't hosted on my main GitHub account, since I wanted to keep everything seperate. I don't host these on an Organization (it's like a mini-account), since those only have 500MB of disk space. For reference, the entire ModMyi repository is 1.1GB, so it runs on an account since those get more space. Each rehost on ck19r is its own GitHub repository, since each one also takes up around 0.5-1GB.
